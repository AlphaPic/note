### 一、开始考试
#### 两调一系
```
1、调左右反光镜(后门把手离上边缘1/4处，离车内1/3处)
2、将座椅调至双脚可以全部踩下而又不费力的位置
3、系安全带
4、为了防止溜车，需要先放离合，等车子处于震动的状态之后再放刹车
```
#### 进考场
```
1、往前走至路角处，右打死转弯
2、待第一步转成和路平行之后，肩部到达铁管时左打死，行至超出路上铁管的位置停车，并进行倒桩[1](
   转弯的时候有点晚)
```
### 二、倒桩
#### 右边倒桩
```
0、先放离合至震动，然后完全放开刹车
1、肩部和左边铁管边缘齐平时，方向盘往右打死
2、当后车轮压到从左到右的最后一根虚线的最左边时，回半圈，如果后轮压线之后没有到最后一根虚线的
   最左边时，表明回早了，此时需要多回一点方向盘，如果后轮没有压线就已经超过了最后一根虚线的最右
   边时，那么表明回晚了，这种情况下已经无法挽救了，只能不回方向盘，并看看一直往后走自己的运气如
   何
3、当车子和左右两边的车库线平行的时候，回正方向盘
4、当自己的眼睛和左边的白杆以及最左边的白色的树桩齐平时，踩刹车[1](有点早了,出来的时候压线压
   晚了)
```
#### 左边倒桩
```
0、先放离合至震动，然后完全放开刹车
1、肩部和左边铁管边缘齐平时，方向盘往左打死
2、当后车轮压到从左到右的第一根虚线的最右边时，回半圈，如果后轮压线之后没有到第一根虚线的最右边
   时，表明回早了，此时需要多回一点方向盘，如果后轮没有压线就已经超过了第一根虚线的最右边时，那
   么表明回晚了，这种情况下已经无法挽救了，只能不回方向盘，并看看一直往后走自己的运气如何
3、当车子和左右两边的车库线平行的时候，回正方向盘
4、当自己的眼睛和左边的白杆以及最左边的白色的树桩齐平时，踩刹车
5、出车库，进行侧方位停车
```
### 三、侧方位停车
```
1、在指示牌之前将左边的反光镜后门把手调至离上边缘1/2处，离车内1/3处
2、金线对准箭头，往前方缓慢行进至C2停车库
3、当肩部和黑色的树齐平时停车
4、挂倒挡，当肩部和白色的树齐平时往右打死
5、看到白色的杆子时，回正方向盘[1](应该是墙角，不是白色的杆子)
6、将头探出窗外，当后轮和车库的虚线相交时，往左打死方向盘
7、往后行进直到车子进入车库，并且和左边的车库线平行时踩刹车[1](前轮没有完全进来，车子没有完
   全和虚线平行)
8、打左方向灯，挂前进挡行进至裂缝离工作台右边1/3处时回正，并向右打一圈方向盘[1](应该是看到
   墙角之后右打死，另外离合有点没有控制住，莫名其妙就松开了)，此时侧方位停车完成，开始进行爬
   坡
```
### 四、爬坡
```
1、爬坡前将大灯打开
2、先放离合至抖动，然后放开刹车前进
3、到达转弯处，右打死转弯，开始上坡，并且视线往左边看
4、当视线和最后一根杆子以及红牛罐子齐平时，同时踩刹车和离合、稍后拉手刹[1](停早了，晚点没关
   系，但是早比较致命，另外和右边的边线隔的距离稍远，应该小于30cm)
5、过3秒之后，放开手刹、放开离合至震动，放开刹车开始前进[1](这里熄火两次,是离合放的不到位，
   导致动力不足熄火的)，开始进入隧道
```
### 五、进出隧道
```
1、隧道是从一个下坡进入的，因此要注意通过踩刹车来控制速度
2、进入隧道之后，当汽车行进至倒数第四根的柱子时，将大灯关闭[1](关的有点晚，超出了倒数第4根柱
   子，另外左边貌似有点压线，这个没有仔细观察)
3、出隧道时，将紧急灯打开，当车子开到路的尽头之后，进行180度的转弯，开始进入紧急路况车道
```
### 六、进出紧急路况车道
```
1、在这里缓慢行驶，当车子里传出前方紧急路况的时候，立刻双脚踩住刹车和离合制动[1](重踩容易熄
   火，因此轻踩就行了*)
2、等待3秒之后，关闭紧急灯，放开离合至震动，然后放开刹车，开始进入S型弯道
```
### 七、进出S型弯道
```
1、将左右两边的反光镜调至后边的门把手紧贴上边缘处，离车内1/3处
2、车子的金线和前方的黑色的树对齐，当车子的头部差不多全部盖住黄标时，往左打一圈方向盘，如果车
   子稍微偏右，那么再向左打多一点方向盘调整一下[1](这里出现了离合没有控制住，发生了失速的情
   况，车速略微有点快)
3、当车子的头部差不多盖住前方的黄标时，往先回正方向盘，然后往右打一圈方向盘，这时候不要多回方
   向盘，等车子出了S型弯道时，往左打死，然后进入直角掉头[1](这里右走1圈的时候回方向回多了，
   导致差点压线，按道理应该不需要回超过1圈多，只需要1圈就行了)
```
### 八、直角掉头
```
1、保持车身的金线和正对面的黄标对齐[1](一开始没有出现对齐的情况，这个是疏忽)
2、当车门中间和最后一个黄标齐平的时候，往左打死方向盘，等车身正了之后，回正方向盘，并且经过接
   下来一个转弯之后，在标牌处方向盘往右打死，开始进入窄路掉头
```
### 九、窄路掉头
```
1、车子的金线和路上的黄点相齐平，缓慢前进
2、当车子距离前边线大概50公分的时候，往左打死方向盘，探出脑袋，等待车子的前轮和边线相隔大概50公
   分的时候，踩刹车制动停车
3、往右打死方向盘，然后挂倒挡，放开离合至震动，然后放开刹车，当后轮和边线相隔大概50公分的时候，
   刹车制动
4、往左打死方向盘，挂前进挡，放开离合至震动，然后放开刹车开始前行，行驶至和边线平行处，回正方向
   盘，并在前方拐角处转弯，开始进入停车拿卡[1](没有太多问题，但是依然出现了离合没有控制住的
   情况)
```
### 十、停车拿卡
```
1、缓慢行驶车辆，并且将头探出窗外，保持和左边线大概30公分左右的宽度，不要超过50公分，否则不
   及格
2、待身体的肩部到达停车拿卡的点时，刹车制动，拉手刹，挂空挡[1](忘记拉手刹，挂空挡了)，左手伸
   出窗外，按一下停车取卡的按钮，然后等待3秒之后，放开离合至震动，然后放开刹车开始前行
```
### 十一、结束考试
```
1、开着车子经过一次右转弯到达考试的开始地点时，考试结束，如果前面所有的步骤都做对了，那么科目二
   通过
```
